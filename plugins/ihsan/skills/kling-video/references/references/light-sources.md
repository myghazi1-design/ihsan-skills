# Named Light Sources

Replace abstract lighting descriptions with specific named sources. Kling 3.0 was trained on cinematography-captioned footage where lights are described by what they physically are.

## Substitution Rule

Never output these terms without a specific replacement:
- ❌ "dramatic lighting"
- ❌ "cinematic lighting"
- ❌ "moody lighting"
- ❌ "beautiful lighting"
- ❌ "atmospheric lighting"
- ❌ "warm lighting" (use a named warm source)
- ❌ "cool lighting" (use a named cool source)

## Light Source Library

### Natural Light

| Source | Visual character |
|---|---|
| **Golden hour sunlight** | Warm, low-angle, long shadows, dust particles visible |
| **Blue hour** | Post-sunset, cool blue tones, soft even illumination |
| **Harsh midday sun** | Top-down, deep shadows, high contrast |
| **Overcast daylight** | Soft, diffused, even, no shadows |
| **Moonlight** | Cool blue-silver, low intensity, long shadows |
| **Dawn mist** | Warm backlit fog, atmospheric depth |
| **Dappled forest light** | Moving light patterns through canopy |
| **Window light (side)** | Soft directional, flattering for faces |
| **Skylight** | Overhead diffused, architectural feel |

### Urban Night

| Source | Visual character |
|---|---|
| **Neon signs** | Magenta, cyan, pink saturated colors, buzzing quality |
| **Flickering fluorescent tubes** | Cold clinical, slight flicker, green tint |
| **Street lamps (sodium vapor)** | Warm orange pools, harsh circles on pavement |
| **LED streetlights** | Cool white, sharp shadows |
| **Car headlights** | Directional beams, lens flare potential |
| **Store window displays** | Colorful interior lighting spilling outside |
| **Traffic lights** | Red/green/amber cycles, reflective on wet surfaces |
| **Phone/screen glow** | Soft blue-white on faces, close-up |

### Interior / Practical

| Source | Visual character |
|---|---|
| **Pendant lamp** | Warm pool from above, rest falls to shadow |
| **Table lamp** | Intimate warm glow, close proximity |
| **Candle light** | Warm orange flickering, very soft |
| **Fireplace** | Dancing warm light, moving shadows |
| **Kitchen overhead fluorescent** | Even practical, shadowless |
| **Bedside reading lamp** | Warm directional, single source |
| **TV glow** | Shifting colors on faces in dark room |
| **Monitor glow** | Cool blue-white, stationary on face |
| **String lights** | Multiple warm points, festive |
| **Exit sign** | Red ambient in dark spaces |

### Cinematic Practical

| Source | Visual character |
|---|---|
| **Studio key light** | Main directional light source |
| **Rim light from behind** | Outlines subject against dark background |
| **Backlight** | Silhouette or halo effect |
| **Underlight** | From below, dramatic/horror |
| **Top light** | From directly above, sharp down-shadows |
| **Practical lamp in frame** | Motivates scene lighting visibly |
| **Anamorphic lens flare** | Horizontal streaks from bright points |
| **Spotlight** | Narrow beam, theatrical |

## Assembly Examples

Scene layer with proper light source integration:

> "Dim 1920s speakeasy at 2 AM. **Low-hanging pendant lamps** cast warm amber pools over wood tables. **Cigarette smoke** drifts through the beams. Rest of the room falls to shadow."

> "Modern tech office at dusk. **Cool LED panel lighting** from overhead fixtures. **Monitor glow** from a laptop illuminates the subject's face with pale blue. **City light** bleeds through floor-to-ceiling windows behind."

> "Forest at golden hour. **Low-angle sunlight** cuts through the canopy creating **dappled light patterns** on the ground. **Dust particles** hang in the light beams. Deep shadows between the trees."

> "Empty parking garage at 3 AM. **Flickering fluorescent tubes** buzz overhead with slight green tint. **Single exit sign** casts red ambient into the far corners. Concrete absorbs most of the light."

## Film Stock References

When the user wants a specific look, reference actual film stocks:

- **35mm film, shallow focus** — warm, grainy, shallow depth
- **Super 8 film** — vintage warm tones, heavy grain
- **VHS camcorder** — heavy grain, chromatic aberration, low fidelity
- **Digital cinema, anamorphic** — modern, sharp, horizontal lens flares
- **Kodak Portra** — flattering skin tones, warm neutral palette
- **Kodak Vision3 250D** — daylight balanced, cinematic contrast
